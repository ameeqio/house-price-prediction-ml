# 🏡 House Price Prediction

An end-to-end Machine Learning project that predicts house prices using the Ames Housing Dataset.

## 📌 Project Overview

This project demonstrates the complete Machine Learning workflow, from raw data to deployment. The goal is to build a regression model capable of predicting house prices based on various property features.

## 🚀 Features

- Exploratory Data Analysis (EDA)
- Data Cleaning
- Feature Engineering
- Model Training
- Hyperparameter Tuning
- Model Evaluation
- Model Serialization
- Streamlit Web Application
- Deployment

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Streamlit
- Git & GitHub
